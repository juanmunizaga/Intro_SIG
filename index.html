Proyección y Corte de información espacial

A continuación, se realizan los principales procedimientos para el manejo de información espacial. Estos procedimientos permiten verificar el sistema de referencia espacial y transformar hacia otro sistema de referencia como pueden ser cambios entre coordenadas geográficas y coordenadas proyectadas, este procedimiento es llamado "Proyectar". Por otro lado, el procedimiento "Clip", permite cortar información espacial entre vectores. De manera que podemos ajustar la información a un área específica que se quiera trabajar.
